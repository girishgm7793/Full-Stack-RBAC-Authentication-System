# Full Stack Authentication & RBAC System

Tech Stack
Backend:
- Spring Boot
- Spring Security
- JWT
- Hibernate

Frontend:
- React
- TypeScript
- TailwindCSS

Features
- User Registration
- Login with JWT Authentication
- Role Based Access Control (USER / ADMIN)
- Protected API endpoints
- Role-based UI rendering

Run Project

Backend
mvn spring-boot:run

Frontend
npm install
npm run dev
